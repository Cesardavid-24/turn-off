# Turn-off
## Simple app to shut-down my pc 

## Run server
to run the server

- **Step 1**
 type on console 
```
cd server
```
- **Step 2**
```
npm install 
```
- **Step 3**  
```
npm run dev
```

## Client Side  
install live server to start the client

## Compile C++
- **Step 1**  
```
cd shut-down
```
- **Step 2**
```
g++ main.cpp
```
you can add -o **<name>** to name the build

## Remember 
- **change the url from localhost to your private ip from your pc where the request is sent**
